Este sistema permite al usuario gestionar un inventario de productos utilizando una base 
de datos SQLite.

Las operaciones que se pueden realizar son: registrar productos, visualizar los productos 
registrados y sus datos, actualizar el inventario, eliminar productos cargados, listar los 
productos y realizar reportes de bajo stock.

Estas opciones se encuentran en el menú principal del sistema, ordenadas de la siguiente 
manera: 

1 - Registrar producto: añadir un nuevo producto al inventario.
2 - Visualizar productos: consultar información de productos en el inventario.
3 - Actualizar producto: modificar el stock de un producto existente.
4 - Eliminar producto: eliminar un producto del inventario.
5 - Listar todos los productos: mostrar todos los productos almacenados.
6 - Reportar bajo stock: generar un reporte de productos con stock bajo un límite definido.
7 - Salir: finalizar el programa.

Cuando el usuario ejecuta el programa por primera vez, se crea la tabla productos.
Luego, siguiendo las instrucciones que el sistema muestra en la pantalla puede seleccionar 
qué acción desea realizar. Las mismas están numeradas del 1 al 7 y podrá acceder a ellas 
ingresando el número que corresponda según el caso.

El sistema mostrará mensajes de error cuando el usuario intente realizar acciones que no 
están disponibles o ingrese datos que no sean válidos.
